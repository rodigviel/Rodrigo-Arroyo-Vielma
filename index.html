Presentación personal

Soy un profesional en formación con especialización en análisis de datos. Me interesa aplicar la tecnología para resolver problemas reales y generar impacto positivo. Proyectos

Descripción: Una aplicación para registrar libros, préstamos y usuarios, además de mostrar resúmenes de los préstamos realizados.

Objetivos: Gestionar y visualizar de forma eficiente la información de una biblioteca.

Tecnologías utilizadas: Python.

Código fuente: Ver en GitHub

Reflexión: Aprendí a manejar bases de datos, gestionar relaciones entre tablas y desarrollar funcionalidades completas para usuarios y préstamos.

Objetivos: Explorar, limpiar y analizar datos de manera interactiva.

Tecnologías utilizadas: Python, Google Colab, Pandas, Matplotlib.

Código fuente: Ver en GitHub

Reflexión: Mejoré mis habilidades en análisis exploratorio de datos y en la comunicación de resultados mediante gráficos claros.

Tecnologías utilizadas: Python, SQLite.

Código fuente: Ver en GitHub
